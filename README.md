# Telegram-бот для "Компьютерной Академии ТОП"

Этот проект представляет собой Telegram-бота, разработанного для студентов "Компьютерной Академии ТОП". Бот предоставляет удобный доступ к учебной информации, такой как расписание и оценки, прямо в мессенджере.

## ✨ Возможности

- 🔐 **Авторизация**: Безопасный вход в систему с использованием логина и пароля от личного кабинета.
- 📅 **Расписание**: Просмотр расписания занятий на выбранную дату.
- 📊 **Оценки**: Просмотр оценок за выбранную дату.
- 📱 **Веб-приложение (Mini App)**: Интегрированное веб-приложение для просмотра расписания.
- 🔒 **Приватность**: Автоматическое удаление сообщений с логином и паролем после авторизации.

## 🛠️ Технологический стек

- **Бэкенд**: C++17, [TgBot-cpp](https://github.com/reo7sp/tgbot-cpp), [CPR](https://github.com/whoshuu/cpr), [nlohmann/json](https://github.com/nlohmann/json).
- **Фронтенд (Mini App)**: TypeScript, Vite.
- **База данных**: SQLite.
- **Сборка и контейнеризация**: Docker, Docker Compose, CMake.
